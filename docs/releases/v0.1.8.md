# v0.1.8

This release cleans up first-launch installer output.

## What changed

- The Windows launcher now sets `UV_LINK_MODE=copy` before running `uv sync` or `uv run`.
- This avoids uv's cross-drive hardlink warning when the runtime cache is on `D:\codexDownload\douyinclaude-runtime` and the installed app is under the current user's `AppData\Local\Programs` directory.
- Runtime downloads and caches still stay under `D:\codexDownload\douyinclaude-runtime`.

## Install notes

- The installer is still unsigned. Windows SmartScreen may warn on first download. 当前安装包仍然未签名。
- First launch can download Python dependencies, Playwright browser assets, and local model files. 首次启动可能需要较长时间。
- Runtime downloads and caches remain under `D:\codexDownload\douyinclaude-runtime`.
- Use `uv run recall stop` when you want to stop the local background web service after development, testing, or before reinstalling.
- Maintenance remains available at `/maintenance`.
