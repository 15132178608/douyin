# Douyin Recall v0.1.5

这一版是 `v0.1.4` 之后的发布体验补丁，重点让安装包说明和首启提示更清楚。

## 主要变化

- GitHub Release 会优先读取 `docs/releases/<tag>.md` 作为正式发布说明。
- 安装包首启脚本会明确提示：当前安装包未签名，Windows SmartScreen 可能提示风险。
- 安装包首启脚本会明确提示：首次启动会下载 Python 依赖和 Playwright 浏览器。
- 运行时下载和缓存继续固定在 `D:\codexDownload\douyinclaude-runtime`。
- 保留 `v0.1.4` 的维护中心、恢复、诊断包和服务生命周期能力。

## 安装包说明

- 下载附件 `DouyinRecallSetup.exe` 后双击安装即可。
- 当前安装包未签名，Windows SmartScreen 可能提示风险；请只使用本 GitHub Release 页面下载的安装包。
- 首次启动会下载 Python 依赖和 Playwright 浏览器，耗时取决于网络。
- 首次生成搜索索引时会下载本地模型，耗时也会比较长。
- 本地 Web 服务默认运行在 `http://127.0.0.1:8000`。

## 常用维护命令

```powershell
uv run recall status
uv run recall stop
uv run recall diagnose
```

日常维护入口：

```text
http://127.0.0.1:8000/maintenance
```

## 数据安全

- 安装包不会包含你的 `.env`、SQLite 数据库、浏览器登录态或用户数据。
- 诊断包只包含脱敏后的环境、服务、任务和日志摘要。
- 卸载程序不会自动帮你批量删除个人数据；需要清理时请先确认备份。
