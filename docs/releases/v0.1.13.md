# v0.1.13

This release makes Windows installs easier to back up before long-term use and safer to upgrade.

## What changed

- Added `Douyin Recall Backup Now` to create a SQLite backup from the Start Menu.
- Added `Douyin Recall Backups` to open `data\exports`.
- Added `Douyin Recall Restore Center` to open `/maintenance` for backup validation and confirmed restore.
- Added a best-effort installer pre-install database copy:
  - source: `data\recall.db`
  - destination: `data\exports\pre-install-recall-*.db`
- `Douyin Recall Control` now includes backup, backups directory, and restore center menu actions.
- `Douyin Recall Control` still prints the 状态摘要 before the menu.
- Restore is still not one-click from Start Menu. It still requires `/maintenance` validation and explicit confirmation.

## Install notes

- The installer is still unsigned. Windows SmartScreen may warn on first download. 当前安装包仍然未签名。
- First launch can download Python dependencies, Playwright browser assets, and local model files. 首次启动可能需要较长时间。
- Runtime downloads and caches remain under `D:\codexDownload\douyinclaude-runtime`, with `UV_LINK_MODE=copy` for cross-drive installs.
- The launcher still runs 启动前健康检查 before preparing dependencies.
- The Start Menu 控制入口 remains `Douyin Recall Control`.
- Existing local operations remain available: `Douyin Recall Stop Service`, `Douyin Recall Health Check`, and `Douyin Recall Repair State`.
- Use `Douyin Recall Backup Now` before risky changes or manual testing.
- Use `Douyin Recall Backups` to inspect `data\exports`.
- Use `Douyin Recall Restore Center` to recover from a validated backup through `/maintenance`.
- Use `uv run recall stop` or `Douyin Recall Stop Service` when you want to stop the local background web service.
- Maintenance remains available at `/maintenance`.
