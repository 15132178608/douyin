# v0.1.10

This release adds a local Windows control entry for installed personal use.

## What changed

- Added a Start Menu 控制入口: `Douyin Recall Control`.
- Added direct Start Menu shortcuts:
  - `Douyin Recall Status`
  - `Douyin Recall Stop Service`
  - `Douyin Recall Maintenance`
  - `Douyin Recall Diagnostics`
  - `Douyin Recall Logs`
- `Douyin Recall Maintenance` opens `/maintenance` directly. If the local web service is not responding, it starts the normal launcher first.
- The control script reuses the same runtime cache settings as the launcher: `D:\codexDownload\douyinclaude-runtime` and `UV_LINK_MODE=copy`.
- The launcher still runs 启动前健康检查 before preparing dependencies.

## Install notes

- The installer is still unsigned. Windows SmartScreen may warn on first download. 当前安装包仍然未签名。
- First launch can download Python dependencies, Playwright browser assets, and local model files. 首次启动可能需要较长时间。
- Runtime downloads and caches remain under `D:\codexDownload\douyinclaude-runtime`.
- Use `Douyin Recall Stop Service` or `uv run recall stop` when you want to stop the local background web service after development, testing, or before reinstalling.
- Maintenance remains available at `/maintenance`.
