# v0.1.7

This release adds a safer update-check experience for the local Windows tool.

## What changed

- Added `uv run recall update` to print the current local version, latest GitHub Release, and `DouyinRecallSetup.exe` download link.
- Added a read-only version/update card to `/maintenance`.
- Update checks use the GitHub Release API and cache the result for one hour so the polling maintenance page does not repeatedly hit GitHub.
- Network failures are non-fatal: the command and maintenance center show the error and keep the rest of the tool usable.

## Install notes

- The installer is still unsigned. Windows SmartScreen may warn on first download. 当前安装包仍然未签名。
- First launch can download Python dependencies, Playwright browser assets, and local model files. 首次启动可能需要较长时间。
- Runtime downloads and caches remain under `D:\codexDownload\douyinclaude-runtime`.
- Use `uv run recall stop` when you want to stop the local background web service after development, testing, or before reinstalling.
- The update checker never auto-installs. It only shows the Release page and installer asset.
- Maintenance remains available at `/maintenance`.
