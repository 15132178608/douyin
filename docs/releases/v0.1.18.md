# v0.1.18

This release makes Windows first-launch failures easier to understand and retry.

## What changed

- The Windows launcher now records the current startup step and prints `失败阶段`, `可能原因`, and `建议下一步` when startup fails.
- Stage-specific hints now cover `uv sync`, `playwright install chromium`, `recall init-db`, and `recall serve`.
- `Douyin Recall Prepare Runtime` now prints `Prepare failed at step:`, `Likely cause:`, and `Recommended next step:` before returning an error.
- Failure output points to `uv run recall diagnose`, runtime cache, startup logs, service logs, `Douyin Recall Health Check`, and `Douyin Recall Repair State`.
- The prepare action remains safe to retry and does not start the local web service.

## Install notes

- The installer is still unsigned. Windows SmartScreen may warn on first download. 当前安装包仍然未签名。
- First launch can download Python dependencies, Playwright browser assets, and local model files. 首次启动可能需要较长时间。
- Runtime downloads and caches remain under `D:\codexDownload\douyinclaude-runtime`, with `UV_LINK_MODE=copy` for cross-drive installs.
- The launcher still runs 启动前健康检查 before preparing dependencies.
- The Start Menu 控制入口 remains `Douyin Recall Control`, and it still prints the 状态摘要 before the menu.
- Existing 健康检查 behavior remains available through `Douyin Recall Health Check`.
- Use `Douyin Recall Prepare Runtime` when first launch fails during uv setup, `uv sync`, Playwright Chromium install, or database initialization.
- `Douyin Recall Prepare Runtime` does not start the local web service, so it can be used as a safe retry step before opening the app.
- Existing local operations remain available: `Douyin Recall Stop Service`, `Douyin Recall Health Check`, and `Douyin Recall Repair State`.
- Backup and recovery entries remain available: `Douyin Recall Backup Now`, `Douyin Recall Backups`, `Douyin Recall Restore Center`, and `Douyin Recall Verify Backup`.
- Use `Douyin Recall Verify Backup` or `recall verify-backup` for a read-only recovery drill against the latest `data\exports` backup.
- Installer upgrade safety backups still use `pre-install-recall-*.db`.
- Use `uv run recall stop` or `Douyin Recall Stop Service` when you want to stop the local background web service.
- Use `Douyin Recall Account Recovery` when `/maintenance` says the Douyin login state may be expired, or open `/auth` directly.
- Maintenance remains available at `/maintenance`.
