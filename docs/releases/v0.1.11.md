# v0.1.11

This release makes the Windows control entry more useful before you choose an action.

## What changed

- `Douyin Recall Control` now prints a 状态摘要 before the menu.
- The summary shows:
  - installed version
  - local service state
  - `/maintenance` URL
  - logs directory
  - runtime cache directory
  - the right start or stop entry for the current state
- `Douyin Recall Status` also prints the same summary before running `uv run recall status`.
- The summary reads local files directly: `pyproject.toml`, `.env`, and `data\runtime\server.json`. This keeps basic status visible even if uv has not been prepared yet.

## Install notes

- The installer is still unsigned. Windows SmartScreen may warn on first download. 当前安装包仍然未签名。
- First launch can download Python dependencies, Playwright browser assets, and local model files. 首次启动可能需要较长时间。
- Runtime downloads and caches remain under `D:\codexDownload\douyinclaude-runtime`, with `UV_LINK_MODE=copy` for cross-drive installs.
- The launcher still runs 启动前健康检查 before preparing dependencies.
- The Start Menu 控制入口 remains `Douyin Recall Control`.
- Use `Douyin Recall Stop Service` or `uv run recall stop` when you want to stop the local background web service after development, testing, or before reinstalling.
- Maintenance remains available at `/maintenance`.
